===================
 CgmRadio Changelog
===================

2.1 (05.01.2019)
----------------
- Show current title during listening 

2.0 (16.03.2014)
----------------

- Added parser for *.pls playlists.
- Integration with Odyssey Web Browser possible.
- Stdlib is no used anymore, so the binary is 6 kB smaller in spite of new
  features.
- List of streams is now allocated in the task pool and freed automatically.
- Source code released on two-clause BSD licence.
