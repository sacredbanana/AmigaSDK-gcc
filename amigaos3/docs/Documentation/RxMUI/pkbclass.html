Pkb class

The Pkb class is used to create piano keyboard images in a virtual group.

Pkb class is a subclass of Pkb.mcc, so you must have Pkb.mcc in MUI:Libvs/MUI to use it. Pkb.mcc is copyright of Calogero CALI'.
