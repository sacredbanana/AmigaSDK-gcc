IFList class

The IFList class is used to create a list of the available communication interfaces.

Anytime a IFList object is going to be rendered in a Listview, the interfaces list is re-read.

IFlist can only be used with Listview objects.

The best performances of IFlist are obtained with Miami.

See List Listview

 

Attributes
Name Type Class Note
Inet N I If set to 0, all the interfaces found are shown.
If omitted or set to 1, just UP interfaces are shown.
