NFloattext class

The NFloattext class is used to show a list made of a ascii text.

A NFloattext object can only be added to a NListview object at NListview object creation time.

NFloattext class is a subclass of NFloattext.mcc, so you must have NFloattext.mmc in MUI:Libs/MUI to use it. NFloattext.mcc is copyright of Gilles Masson.

See NList NListview.
