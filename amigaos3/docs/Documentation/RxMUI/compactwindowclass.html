CompactWindow class

CompactWindow class is used to create mini tiny windows.

CompactWindow class is a subclass of CompactWindow.mcc, so you must have CompactWindow.mcc in MUI:Libs/MUI to use it. CompactWindow.mcc is copyright of Szymon Ulatowski.
