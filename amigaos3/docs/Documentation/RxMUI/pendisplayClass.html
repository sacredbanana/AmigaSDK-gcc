Pendisplay class

The Pendisplay class is used to display a pen specification.

 

Attributes
Name Type Class Note
PDSpec S ISGN A string:
  • m<x> where <x> is a MUI pen backgrond from 0 to 7
  • p<x> where <x> is a colormap -128 to 127
  • r<red>,<green>,<blue> where <red>,<green>,<blue> are the 8 digits RGB hex component of a color specification
