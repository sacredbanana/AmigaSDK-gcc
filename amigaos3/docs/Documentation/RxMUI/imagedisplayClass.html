Imagedisplay class

The Imagedisplay class is used to display an image specification.

 

Attributes
Name Type Class Note
IDFixed B I Make the size of the image in the button fixed.
IDSpec S ISGN A string:
  • 0:<x> where <x> is a MUI pattern backgrond from 128 to 145
  • 1:<x> where <x> is a MUI builtin image from 0 to 23
  • 2:m<x> where <x> is a MUI pen backgrond from 0 to 7
  • 2:p<x> where <x> is a colormap -128 to 127
  • 2:r<red>,<green>,<blue> where <red>,<green>,<blue> are the 8 digits RGB hex component of a color specification
  • 3:<x> where <x> is an external image name of max 61 chars (MUI bug)
  • 5:<x> where <x> is a file name of max 61 chars (MUI bug)
