I like MUI and I made this library to use the MUI system in ARexx macros.

This means you can benefit of the MUI programming philosophy, which is one of the best thing happened to Amiga so far, in your ARexx macros.

RxMUI will also introduces you in the Object Oriented Programming with a simple interpreted language as ARexx is.

RxMUI e-mails discussion list is hosted by YahooGroups.

 
Warning: This guide does not describe any attribute that has the same meaning of the original MUI class one. Refer to the MUI developers autodocs for that. You may download them now.
