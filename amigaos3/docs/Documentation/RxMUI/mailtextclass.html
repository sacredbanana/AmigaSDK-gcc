Mailtext class

The Mailtext class is used to display e-mail text.

Mailtext class is a subclass of NList class and can only be added to NListview object .

Mailtext class is a subclass of Mailtext.mcc, so you must have Mailtext.mcc in MUI:Libs/MUI to use it. Mailtext.mcc is copyright Olaf Peters.

See NList NListview.
