Icon class

The Icon class is a subclass of Area class and it is used to display icons.

Of course, you may give an Icon object a Frame, a Background and an Inputmode, to make it act like an image button.

Icon is a subclass of Icon.mcc so you must have Icon.mcc in MUI:Libs/MUI to use it. Icon.mcc is copyright of Russell Leighton.
